# All in Davao - Tourism Platform
is a full-stack single page application (SPA) designed to showcase all the things Davao can offer. Built with an Angular frontend and an ASP.NET Core Web API backend.

## Stack

- ** Framework & Language **: Angular, TypeScript, C#, ASP.NET Core Web API
- ** Markup & Styling **: HTML5, CSS3
- ** Libraries **: RxJS
- ** Database **: SQL Server Express, Entity Framework (EF) Core
- ** Tools **: Swagger, Postman, SQL Server Management Studio (SSMS)
